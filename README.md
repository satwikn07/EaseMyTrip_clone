# Easemytrip
> EaseMyTrip is an Indian online travel company, founded in 2008 by Nishant Pitti, Rikant Pitti, and Prashant Pitti. Headquartered in New Delhi, the company provides hotel bookings, air tickets, holiday packages, bus bookings, and white-label services. EaseMyTrip has overseas offices in Singapore, Dubai, Maldives, and Bangkok.

## Installing and Running
> Executing the commands mentioned below one by one in your command line will help in making this React app up and running on your system in the development mode.
> 
> ```bash
> git clone https://github.com/satwikn07/EaseMyTrip_clone.git
> cd ./Easemytrip
> npm install
> npm run server
> npm start
> ```
> After entering the commands mentioned above, open [http://localhost:3000](http://localhost:3000) to view the app in the browser.
> The backend will run on port 8000
